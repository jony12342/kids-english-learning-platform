## 📝 使用示例

### 代码中引用素材

```typescript
// ✅ 正确：引用静态图片
import starImage from '@/public/images/ui/star_gold.png';

<Image src={starImage} alt="Star" width={64} height={64} />

// 或者
<Image src="/images/ui/star_gold.png" alt="Star" width={64} height={64} />

// ✅ 正确：播放音效
const playSound = (soundFile: string) => {
  const audio = new Audio(`/sounds/effects/feedback/${soundFile}`);
  audio.play();
};

playSound('correct_star.mp3');

// ✅ 正确：背景音乐循环播放
const bgMusic = new Audio('/sounds/background/garden_main.mp3');
bgMusic.loop = true;
bgMusic.volume = 0.3;
bgMusic.play();
```

## ⚠️ 重要注意事项

1. **版权问题**：
   - 确保所有素材可商用
   - 优先选择CC0或CC-BY协议
   - 保留素材来源信息

2. **文件大小**：
   - 图片：单张不超过500KB
   - 音频：单首不超过5MB
   - 总素材大小建议不超过50MB

3. **命名规范**：
   - 使用小写字母和下划线
   - 不要使用空格和特殊字符
   - 使用描述性文件名

4. **格式要求**：
   - 图片：PNG（透明背景）或JPG（不透明）
   - 音频：MP3（音乐）或WAV（音效）
   - 动画：JSON（Lottie）

---
